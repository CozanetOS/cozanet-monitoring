# cozanet-monitoring

Core system monitoring and metrics tracking utilities for the Cozanet OS ecosystem.

## Features

- **Metrics Collection**: Track, query, aggregate, and flush metrics to Supabase database.
- **Health Monitoring**: Flexible active-checking registration and reporting.
- **Pino-based Logger**: Console logging with local memory ring buffer.
- **Performance Timer**: Track sub-operation durations with statistical reporting.
- **Event-driven Alerts**: Trigger/resolve alerts with event emitters.
- **System Diagnostics**: Comprehensive system audits.
